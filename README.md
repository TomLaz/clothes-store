<h2>Stack</h2>
<ul>
<li>React</li>
<li>Typescript</li>
<li>Context API</li>
<li>Material UI</li>
<li>Node</li>
<li>Fake REST API with json-server</li>
<li>JWT authentication</li>
<li>SASS + BEM</li>
<li>CSS Custom Properties (variables)</li>
<li>React Testing Library(Coverage 99%)</li>
<li>Mobile First</li>
<li>Internationalization (i18n)</li>
<li>Linter Code & Style</li>
<li>Lazy Loading</li>
<li>Precommit</li>
<li>Modal</li>
</ul>

<h3>Get Started</h3>
<pre>
git clone https://github.com/TomLaz/clothes-store.git
cd clothes-store
npm install</pre>

<h2>Backend</h2>
<pre>
cd backend
npm run start</pre>

<h2>Frontend</h2>
<pre>
cd frontend
npm run start</pre>
